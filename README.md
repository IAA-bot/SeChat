# SeChat

Lightweight, secure, privacy-respecting P2P encrypted messenger for Android.

- End-to-end encrypted (libsignal X3DH + Double Ratchet)
- Anonymous ephemeral identities (no phone/email)
- Peer-to-peer (no central server)
- LAN discovery via mDNS
- Optional Tor/Orbot routing
- Full local encryption (sqlcipher)

**Status:** MVP in development.
